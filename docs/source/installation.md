# Installation

## Browser

The easiest way to use `dlite` is with a script tag.

```html
<script type="module">
  import Dlite from '//unpkg.com/dlite';
  
  ...
</script>
```

## NPM

You can also install via NPM.

```shell
npm install dlite
```

Then, import `dlite` in your HTML.

```js
import Dlite from 'dlite';
```
